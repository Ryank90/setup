Setup - MAC OS X
=====

Run the commands below to get setup in a jiffy.

__Stack Tools__

Will give you the following:

* Node
* NPM
* Yeoman
* Grunt
* Bower
* SASS
* Xcode CLI Tools
* Homebrew
* Composer

=====

    curl -sSL https://raw.github.com/ryank90/setup/master/setup.sh > /tmp/setup.sh

Then run the following command in terminal

	sh /tmp/setup.sh
